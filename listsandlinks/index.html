Instruction to Create a New Webpage on the production server

  1. Create your 2 files in your respiritory account by typing "gedit main.css" and "gedit index.html".
  2. In the index type the html code for creating the web page, which the format can be found on google.
  3. Go back to the terminal and type "python -m SimpleHTTPServer"
  4. Go back to google, at the url bar, type "localhost:8000", which will enbled you to see your work process that you are working on.
  5. In additional, if you want to add any extra decoration, you can go to main.css folder and type the css code that will make change in your webpage. However, the code can be found in google.
